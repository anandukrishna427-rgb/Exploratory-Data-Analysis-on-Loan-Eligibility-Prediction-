# Exploratory-Data-Analysis-on-Loan-Eligibility-Prediction-

Analyzed 614 loan applicant records from Kaggle to reveal patterns in demographics, finances, and credit factors influencing approval. Includes univariate, bivariate, and multivariate visualizations using Python (Pandas, Matplotlib, Seaborn) to identify approval trends, biases, and key predictors like income and credit history.
